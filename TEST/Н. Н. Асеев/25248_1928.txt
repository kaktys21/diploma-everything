6

Оставьте, / баптисты, / скучную / проповедь, ―
Вам / этих дней / все равно / не отпробовать.
Тот ― / не уныл, / кто горечью / хвалится.
Радость / с луны / все равно / не свалится.
Молотом, / скальпелем, / клапаном, / книгою ―
сердце / по каплям / волнение / двигает.
Сердце мое, / Волнуйся / и стукай!.
Жизнь ― / не очень / понятная / штука.
Сердце мое, / Тревожься / и рвись
вниз, / в глубину, / и ― вверх, / ввысь!
Свет твой / вечный ― / с открытой / душой ―
первой / встречной, / далекой, / чужой;
Шире / и выше / взлета / задор,
Пока / от вспышек / не сгинет / мотор,
пока / не сгаснет / горенья / руда,
пока / от сказки / не станет / следа!
