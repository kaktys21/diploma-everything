НОРВЕЖСКАЯ МЕЛОДИЯ

1

Я с тоской, / Как с траурным котом,
День-деньской / Глядим на старый дом,
До зари / В стакан гремит струя…
(О, Мария / Милая моя…  )

2

Корабли / Сереют сквозь туман.
Моря блик / Сведет меня с ума.
Стой!  Замри, / Скрипичная змея!
(О, Мария / Милая моя…  )

3

Разве снесть / В глазах бессонных соль?
Разве есть / Еще такая боль?
О миры / Скрежещется ноябрь!
(О, Мария / Милая моя…  )

4

Кончен грог.  / Молочницы скрипят.
Скрипку в гроб, / Как девочку, скрипач.
Звонари / Уходят на маяк.
(О, Мария / Милая моя…  )

5

День, как год, / Как черный наговор.
Я да кот, / И больше никого,
Примири / Хоть с гибелью меня,
О, Мария / Милая моя…
