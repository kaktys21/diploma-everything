Ты / Без меня ― / Только дым без огня.
Ты / Без меня ― / Это только одно
Блещущей цепи немое звено,
И не подымет, конечно, оно
Якорь, ушедший на самое дно.
Ты / Продолжалась бы после меня
Только бы разве как ночь после дня
С бледными / Звездами / Через окно.
Эхо мое!  / Но со мной заодно,
Ты повторяешь средь ночи и дня:
«Ты / Без меня ― / Только дым без огня!»
