Приезжают в столицу / смиренно и бойко
молодые Есенины / в красных ковбойках.

Поглядите, / оставив предвзятые толки,
как по-детски подрезаны / наглые челки.

Разверните, / хотя б просто так, / для порядка,
их измятые в дальней дороге / тетрадки.

Там / на фоне безвкусицы и дребедени
ослепляющий образ / блеснет на мгновенье.

Там / среди неумелой мороки
вдруг возникнут / почти гениальные строки.

…  Пусть придет к ним / потом, через годы, по праву
золотого Есенина / звонкая слава.

«Дай лишь бог, ― говорю я, / идя стороною, ―
чтобы им / (извините меня за отсталость)
не такою она доставалась ценою,
не такою ценою она доставалась».
