НА МОГИЛЕ

Н. В. Бугаеву

В лепет / Ясных / Стрекоз
И серебряных бабочек, ―

Точно из слез, ―
Трепет / Красных / Лампадочек;

Громче / Вьюнок / Вьется
С бледной березою;

Звонче / Венок / Бьется
Розовой розою;

Встрепет / Раскрыл
Из кипения / Зелени ―

Мраморы / Крыл
Иссеченного / Гения.

Взвей, / О родной, ―
Из-за времени / Веющий, ―

Взвей / Надо мной
Кисеей / Дымовеющей.
