ШУТ

Баллада

1

Есть край, где старый / Замок
В пучину бьющих / Вод
Зубцами серых / Башен
Глядит ― который / Год!

Его сжигает / Солнце;
Его дожди / Секут…
Есть королевна / В замке
И есть горбатый / Шут!

Докучно / Вырастая
На выступе / Седом, ―
Прищелкивает / Звонко
Трескучим / Бубенцом.

Струею красной / В ветер
Атласный плащ / Летит: ―
На каменных / Отвесах
Подолгу шут / Сидит;

И долго, долго / Смотрит
На запад / Огневой;
В вечерние / Туманы
Колпак подкинет / Свой.

Из каменных / Пробоин
Взлетает стая / Сов,
Когда несется / С башни
Трубы далекий / Рев.
