ГРОЗОВОЕ ОБЛАКО

Блещет откос / Бриллиантами;
Нежно звездеют / Стога; ―

Вздыбились тучи / Гигантами;
Вылились в высь / Жемчуга.

Там, ― в горизонте / Дымеющем,
Ветрами в воздух / Взвитой, ―

Кто-то во вретище / Веющем ―
С молоньей встал / Золотой.

В воздух волнистыми / Гребнями
Плащ его плещется /С плеч, ―

Чтобы над землями / Древними ―
Летними ливнями / Лечь.

По́ небу ухнувшим / Молотом
В хаосы сваленных / Скал ―

Облак, расколотый / В золото, ―
С хохотом он / Оборвал.

Теплится в тающей / Темени
Строгими вздрогами / Глаз: ―

Зреет со светлого / Темени ―
Звездочки летней / Алмаз.

Кутает хмурыми / Бурями
Плечи сутулые / Он; ―

Встанет, овеян / Лазурями, ―
Канет, ― в немой / Небосклон.
