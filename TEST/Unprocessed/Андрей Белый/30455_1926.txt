ИЗ ЛОНА ЛЕТ

И муть / И жуть ― // Из тьмы / Тысячелетий.

И тьмою бед / Разорван / Тусклый гром;

И ― / ― Плачем мы, ― // Потерянные дети…

Очерчен ― / Круг; / Оторван ― / ― Смысл и путь.

Вдруг, ― / Губы / К нам ― / ― В расширенные / Уши, ―

― Как трубы, / Протопыренные / В свет, ―

― Склоняясь там, / Недышащие души, ―

Из лона / Лет ― // Забормотали / Бред.
