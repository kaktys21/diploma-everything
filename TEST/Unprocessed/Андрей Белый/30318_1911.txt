5

Есть край, / Где старый / Замок
В пучину / Бьющих / Вод
Зубцами / Серых / Башен
Глядит ― / Который / Год!

Его сжигает / Солнце,
Его дожди / Секут…
Есть Королевна / В замке
И есть / Горбатый / Шут.

С вершины мшистой / Башни
Гремит в закат / Труба, ―
И над мостом / Чугунным
Мелькает тень / Горба:

То за стеной зубчатой
Докучный бег / Минут / Трещоткой деревянной
Отсчитывает / Шут.

О, королевна, близко
Спасение твое:
В чугунные ворота
Ударилось копье!
