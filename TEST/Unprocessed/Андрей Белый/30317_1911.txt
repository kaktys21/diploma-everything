4

Поток / Рыдает / Пеной,
Клокочет / Бездной / Дней…
В решетчатые окна
Влетает сноп огней.

Расплачется в воротах
Заржавленный засов:
Пернатый / Ясный / Рыцарь
Летит / Из тьмы / Веков.

Конем / Кидаясь / В солнце
Над пенистым ручьем, ―
Гремит трубою в ветер,
Блистает / В даль / Копьем.

Дрожащий / Луч / Играет,
Упав из-за плеча,
Голубоватой сталью
На / Острие / Меча.

И / Бросило / Забрало
Литое серебро
Косматым / Белым / Дымом
Летящее перо.

И / Плещется попона
За / Гривистым конем; ―
― Малиновым, / Тяжелым,
Протянутым / Крылом.
