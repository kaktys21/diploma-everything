ШУТКА

В / Долине / Когда-то / Мечтательно

Перед / Вами / Я, ― // Старый / Дурак, ―

Игрывал / На / Мандолине.

Вы ― / Внимали / Старательно.

И ― / ― Стародавний Зодиак.

Как-то / Избили
И / Выгнали / Меня / Из / Цирка

В / Лохмотьях / И / В / Крови,

Вопиющего ― //― О Боге!  / ― Боге!  / ― Боге!

И / О ― // ― Вселенской любви.

Вы / Случайно / Встретили / Поющего / Паяца ―

Постояли, / Послушали / Пение.

Вы ― / Отметили // Дурацкий / Колпак.

Вы ― / Сказали / Внимательно: // ― «Это― / Путь / Посвящения»…

Вы ― / Мечтательно / Уставились

В ― / ― Зодиак.
