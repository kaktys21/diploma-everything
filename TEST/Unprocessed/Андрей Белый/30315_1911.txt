2

В тяжелый, знойный полдень,
Таясь / В тени / Аркад, ―
Выходит королевна
Послушать / Треск / Цикад.

Из / Блещущих / Травинок,
Из росянистых пней, ―
Небесною коронкой
Цветок / Смеется / Ей.

Едва / Она / Сломала
Высокий стебелек, ―
«О, / Королевна, / Вспомни», ―
Пролепетал цветок;

Едва / Она / Сломала
Высокий стебелек, ―
Кипучею струею
Ей в очи / Брызнул / Сок.

Блестя, запели воды ―
Окрестность, / Луг, / Цветы…
Запел / Старинный / Ветер:
«О, вспомни, вспомни ты!»

Прошел родимый замок,
Как облако над ней: ―
Зубцами / Старых / Башен
Растаял / В бездне / Дней…
