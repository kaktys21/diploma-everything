Ночь.  / Разглагольствовал / месяц.
Он погибал ―
― по надменности польского / акцента ―
― по губам.
Широк, / как тайна
на́ ухо, ― / ― пароль
шпика, / ― бас октавный, ―
плес ночи.  // Порой
шелка соловьев.  / Слушаючи.
Целуя.  / Ночь на ушко.
Порой ночь / ― старушечий
ридикюль средь шажков, ―
подробно, / всеми силами
теряя значенье,
о, / снами выломив
себя, / с моих очей не
спускала любви.
Опять.  / Повторной.

Даря воздух горный.
До задыханья обвив.
