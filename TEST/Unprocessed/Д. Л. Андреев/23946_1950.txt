2. НА БАЛКОНЕ

Островерхим очерком вдали ― / Кремль / синий,
А внизу ― клокочущая хлябь, / Поток:
Пятна перемешивая, смыв / рябь / линий,
Улица, как Волга, бурлит / У ног.

Ветром овеваемы, теплом / дня / полны,
Высотой качаемы, смеясь, / Поем,
Чтоб с закатов розовых неслись / к нам / волны,
Нежа нас над городом ― одних, / Вдвоем.

Зрелища и гульбища уже / чуть / алы,
Пенясь, точно свадебным вином / Ковши…
Ах, дитя беспечное!  Когда б / ты / знала
Яд возревновавшей об одном / Души!

Имя повелительницы всех / снов / духа
Я не прошепчу тебе ни здесь, / Вверху,
Ни ― в полночных комнатах, где плач / твой / глухо
Канул бы, как ветер среди хвой, / Во мху.

Только предвечерием ты кинь / взгляд / чуткий
На лицо любимое, когда / С террас
Вижу, как над городом бурлит / яд / в кубке:
Дымка над громадами труда, / Зыбь трасс;

Иль, когда средь ночи, меж глухих / штор / спальной,
Фонари бесчисленные ― там, / На дне,
Чертят, как узоры на шелках / мглы / дальной,
Имя, приоткрытое судьбой / Лишь / мне.
