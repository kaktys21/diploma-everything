I. ВОЙНА, КОТОРАЯ БУДЕТ СЕЙЧАС

Когда / перелистываем / газетный лист мы,
перебираем / новости / заграницы болотной,
натыкаемся ― / выдумали / ученые империалистовы:
то газ, / то луч, / то самолет беспилотный.
Что́ им, / куриная судьба горька?
Человечеству / помогают, / лучи скрестя?
Нет ― / с поднебесья / новый аркан
готовят / на шеи / рабочих и крестьян.
Десятилетие / страницы / всех газетин
смерть начиняла ― / увечья, / горе…
Но вздором / покажутся / бойни эти
в ужасе / грядущих фантасмагорий.
