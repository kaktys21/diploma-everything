НЕБЕСНЫЙ ЧЕРДАК

Мы пролетали, / мы миновали
местности / странных наименований.
Среднее / между / «сукин сын»
и между / «укуси» ―
Сууксу / показал / кипарисы-носы
и унесся / в туманную синь.
Го- / ра.
Груз.  / Уф!
По- / ра.
Гур- / зуф.
Станция.  / Стала машина старушка.
Полпути.  / Неужто?!
Правильно / было б / сказать «Алушка»,
а они, как дети ― / «Алушта».
В путь, / в зной,
крутизной!
Туда, / где горизонта черта,
где зубы / гор / из небесного рта,
туда, / в конец, / к небесам на чердак,
на ― / Чатырдаг.
Кустов хохол да редкие дерева.
Холодно.  / Перевал.
Исчезло море.  / Нет его.
В тумане фиолетовом.
Да под нами / на поляне
радуги пыланье.
И вот / умолк / мотор-хохотун.
Перед фронтом / серебряных тополей
мы / пронеслись / на свободном ходу
и через час ― / в Симферополе.
