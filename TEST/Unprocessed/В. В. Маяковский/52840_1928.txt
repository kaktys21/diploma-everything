ШЕСТОЙ

Как будто / чудовищный кран
мир / подымает уверенно ―
по ступенькам / стран
подымаются / на конгресс Коминтерна.
Фактом / живым / встрянь ―
чего и представить нельзя!
50[пятьдесят] / огромнейших стран
входят / в один зал.
Не коврами / пол стлан.
Сапогам / не мять, / не толочь их.
Сошлись / 50[пятьдесят] стран,
не изнеженных ― / а рабочих.
Послало / 50[пятьдесят] стран
гонцов / из рабочей гущи,
войны / бронированный таран
обернуть / на хозяев воюющих.
Велело / 50[пятьдесят] стран:
«Шнур / динамитный / вызмей!
Подготовь / генеральный план
взрыва капитализма».
Черный / негр / прям.
Японец ― / желт и прян.
Белый ― / норвежец, верно.
50[пятьдесят] / различнейших стран
идут / на конгресс Коминтерна.
Похода времени ― / стан.
Револьвера дней ― / кобура.
Сошлись / 50[пятьдесят] стран
восстанию / крикнуть: / «Ура!»
Мир / буржуазный, / ляг!
Пусть / обреченный валится!
Колонный зал / в кулак
сжимает / колонны-пальцы.
Будто / чудовищный кран
мир / подымает уверенно ―
по ступенькам / 50[пятьдесят] стран
поднялись / на конгресс Коминтерна.
