III.  ПРИЗЫВ.

Крылатых / дней / далека дата.
Нескоро / в радости / крикнем: / ― Вот они!  ―
Но я ― / грядущих дней агитатор ―
к ним / хоть на шаг / подвожу сегодня.
Чтоб вам / уподобиться / детям птичьим,
в гондолу / в уютную / сев, ―
огнем вам / в глаза / ежедневно тычем
буквы ― / O. Д. В. Ф. [одэвээф]
Чтоб в будущий / яркий, / радостный час вы
носились / в небе любом ―
сейчас / летуны / разбиваются насмерть,
в Ходынку / вплющившись лбом.
Чтоб в будущем / веке / жизнь человечья
ракетой / неслась в небеса ―
и я, / уставая / из вечера в вечер,
вот эти / строки / писал.
Рабочий!  / Крестьянин!  / Проверь наощупь,
что / и небеса ― / твои!
Стотридцатимилльонною мощью
желанье / лететь / напои!
Довольно / ползать, как вошь!
Найдем ― / разгуляться где бы!
Даешь / небо!
Сами / выкропим рожь ―
тучи / прольем над хлебом.
Даешь / небо!
Слов / отточенный нож
вонзай / в грядущую небыль!
Даешь / небо!
