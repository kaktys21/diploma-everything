О. Д. В. Ф.

― Не понимаю я, ―
сказал / один, / в раздумье сев, ―
что это / за такие / воздушные друзья
и что такое / О.  Д.[дэ] В.[вэ] Ф.[эф]?  ―
Товарищ, / брось / в раздумье коптиться!
О.  Д.[дэ] В.[вэ] Ф.[эф] ― / это / тот,
кто сделает / тебя / птицей.
― А на что / воздушный флот?  ― / Вот.
Если / враг / с пулеметом, / враг с картечью
налетит, / на аэро сев, ―
кто / в защиту / ему / полетит навстречу?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / лихорадка, / народ калеча,
налетит, / комаром насев, ―
хину / кто повезет?  / Кто / тебя / излечит?
Самолет.  /А его /построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Кто / прикажет, / а вёрсты увидев рыбу:
«Догоняй, / на лодки сев?»
(Ведь ее / с берегов / и не взвидели вы бы!  )
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / надо / в момент / исследовать местность,
враг / откуда / стреляет, засев, ―
кто / ее промерит?  / Известно:
самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / надо / вмиг / доставить известие,
хоть верхом / на ласточку / сев, ―
кто / его / доставит / с ветром вместе?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / Антанта / на нас / рассердится,
кто / смирит / антантин гнев?
Успокоит / антантино сердце
самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / солнце / огненным глазом
выжигает / крестьянский посев ―
кто / заставит / тучи / раздождиться наземь?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / надо / срочно / доставить клади,
хоть в ковер / в волшебный / засев, ―
кто / и тысячу / верст / отмахает на день?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Когда / жара / и дождь когда,
кто, / на тучи / чуть не сев,
предскажет / крестьянину / и жар / и холода?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если надо, / чтоб мчал / пассажир,
в день / десятки / верст облетев, ―
кто / отмашет / небесную ширь?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Если / не тушит / ни одна душа,
а пожар / разевает зев ―
кто / взлетит / в облака, / туша?
Самолет.  / А его / построит / О.  Д.[дэ] В.[вэ] Ф.[эф]
Вот
для чего / воздушный флот!
Чтобы, бомбы / в кузов / грузя,
не грозили / враги / насев,
объединились / летанья друзья
в содружество / О.  Д.[дэ] В.[вэ] Ф.[эф]
