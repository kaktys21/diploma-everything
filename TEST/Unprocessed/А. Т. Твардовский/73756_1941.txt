КОГДА ТЫ ЛЕТИШЬ

Когда ты летишь / Поутру на работу,
С земли своего / Узнают по полету.

По крыльям знакомым, / По звуку мотора
Тебя узнают / На дороге шоферы.

Тебя провожают / Колхозницы в поле
Напутственным словом: / ― Лети, наш соколик.

Лети, наш родимый, / На славу сражайся,
Живой, невредимый / Назад возвращайся…

И, спинкой мелькнув / Меж подсолнухов голой,
Бежит на задворки / Трехлетний Микола.

По грядкам бежит, / Спотыкаясь, мальчонка,
Он машет тебе / Загорелой ручонкой.

Он долго и жадно / Следит за тобой,
Он тоже тебя / Посылает на бой.

А там, за рекой, / За крутым поворотом,
Тебя уже видит / Родная пехота.

И лица усталые / Сразу моложе,
И если б ты слышал, / Услышал бы тоже:

― Наш славный товарищ, / За взгорком враги.
До них подобраться / Ты нам помоги.

Хвати их фугасом / По каскам литым.
Мы лишнего часу / Им жить не дадим.

Спасибо, товарищ, / За помощь в бою,
Спасибо, родной, / За работу твою…

И вот, развернувшись, / Летишь ты обратно.
Машина работает / Ровно и внятно.

И вновь под тобою ― / Прибрежные села,
И щурится, глядя / На солнце, Микола.

Кричит с огорода: / ― Ой, баба, ой, мама,
Бегите, глядите ― / Тот самый, тот самый!
